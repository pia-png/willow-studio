# Willow Studio

Willow Studio is an interactive typography playground built to showcase **Willow**, my custom handwritten font. The website lets users experiment with it in real time, creating personalized designs and exporting them as images. It is a more developed version of the preview section of the original willow website. 

## Features

* Live text preview
* Adjustable font size
* Letter spacing controls
* Line height controls
* Text color picker
* Background color picker
* One-click design presets
* Random inspirational quote generator
* Download designs as PNG
* Responsive two-panel editor
* Smooth animations and instant updates

## Built With

* HTML5
* CSS3
* JavaScript 

## How It Works

1. Type any text into the editor.
2. Customize its appearance using the control panel.
3. Try different preset styles for instant inspiration.
4. Generate a random quote with one click.
5. Download your finished design as a PNG.

## Why I Built It

I created Willow Studio to showcase my custom font in a more interactive and engaging way than a traditional specimen page. I wanted visitors to experiment with the typeface, explore different design styles, and create something unique while experiencing the personality of the font.
